**step-by-step:**

1. **malfunction occurs**

   - system appends: `{"stone":"X","status":"passed"}`
   - OR: reviewer/judge now passes
   - system appends: `{"stone":"X","status":"passed"}`

4. **hook allows stop**
   - `route.drive --mode hook` checks latest status
   - latest status is "passed"
   - exit code is 0
