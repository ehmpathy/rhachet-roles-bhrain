# product blueprint

## summary

implement route reviewer failsafe to:
1. capture stdout/stderr in tree bucket format for reviewers and judges
2. classify exit codes (0=passed, 2=constraint, other=malfunction)
3. record malfunction status in passage.jsonl
4. halt route.drive on malfunction with escalation message

## codepath tree

### domain.objects

```
PassageReport
├─ [○] status: 'passed' | 'approved' | 'blocked' | 'rewound'
├─ [+] status: ... | 'malfunction'
└─ [○] blocker?: 'review.self' | 'review.peer' | 'judge' | 'approval'
```

note: blocker field unchanged. when status='malfunction', blocker is not needed (status is self-explanatory).

### domain.operations/route/guard

```
getExitCodeClass (new)
├─ [+] input: { code: number }
├─ [+] output: ExitCodeClass = 'passed' | 'constraint' | 'malfunction'
└─ [+] logic:
   ├─ code === 0 → 'passed'
   ├─ code === 2 → 'constraint'
   └─ else → 'malfunction'

formatTreeBucket (new)
├─ [+] input: { label: string; content: string }
├─ [+] output: string (tree bucket format)
└─ [+] logic:
   ├─ emit '├─ {label}'
   ├─ emit '│  ├─'
   ├─ emit '│  │'
   ├─ for each line: emit '│  │  {line}'
   ├─ emit '│  │'
   └─ emit '│  └─'

runStoneGuardReviews
├─ [○] execa command with reject: false
├─ [+] capture exitCode as number
├─ [+] classify via getExitCodeClass
├─ [+] include stdout, stderr, exitCode, exitClass in result
├─ [~] format artifact with formatTreeBucket (replace ---stderr--- format)
└─ [○] return GuardExecutionResult[]

runStoneGuardJudges
├─ [○] execa command with reject: false
├─ [+] capture exitCode as number
├─ [+] classify via getExitCodeClass
├─ [+] include stdout, stderr, exitCode, exitClass in result
├─ [~] format artifact with formatTreeBucket (replace ---stderr--- format)
└─ [○] return GuardExecutionResult[]
```

### domain.operations/route

```
formatRouteStoneEmit
├─ [○] emit summary tree (formatGuardTree)
└─ [+] emit detailed output for FAILED reviews and judges only (exit != 0):
   ├─ 🔎 review N
   │  ├─ formatTreeBucket(stdout)
   │  ├─ formatTreeBucket(stderr)
   │  └─ passage blocked
   │     ├─ blocked by constraints | blocked by malfunction
   │     └─ exit code: N
   └─ 🪶 judge N
      ├─ formatTreeBucket(stdout)
      ├─ formatTreeBucket(stderr)
      └─ passage blocked
         ├─ blocked by constraints | blocked by malfunction
         └─ exit code: N

stepRouteDrive
├─ [○] read passage.jsonl
├─ [+] detect malfunction status
├─ [+] on malfunction:
│  ├─ emit escalation message
│  └─ exit code 3 (immediate halt, not retry)
└─ [○] on constraint: continue to drive, guide agent
```

## contracts

### exit code contract

| exit code | definition | exitClass |
|-----------|------------|-----------|
| 0 | passed | 'passed' |
| 2 | ConstraintError | 'constraint' |
| other | MalfunctionError | 'malfunction' |

### artifact format contract

```
🔎 review N
├─ stdout
│  ├─
│  │
│  │  [stdout content]
│  │
│  └─
├─ stderr
│  ├─
│  │
│  │  [stderr content]
│  │
│  └─
└─ passage blocked
   ├─ blocked by constraints | blocked by malfunction
   └─ exit code: N
```

### passage.jsonl contract

```jsonl
{"stone": "1.vision", "status": "malfunction"}
```

note: blocker field omitted for malfunction. status is self-explanatory.

### route.drive behavior contract

| state | behavior |
|-------|----------|
| constraint failure | continue to fire, guide agent |
| malfunction | immediate halt, exit 3, escalate to human |
