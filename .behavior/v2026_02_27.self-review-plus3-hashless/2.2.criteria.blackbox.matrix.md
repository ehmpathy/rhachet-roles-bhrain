## matrix.1 = promise attempt outcomes

dimensions:
- **ind: glob count** = number of triggered files for this slug (different hashes)
- **ind: $hashbar** = configured threshold (default: 3)
- **ind: mtime** = age of newest triggered file
- **dep: passage** = outcome of --as promised attempt
- **dep: promise type** = what kind of promise is created

| ind: glob count | ind: $hashbar | ind: mtime | dep: passage | dep: promise type |
|-----------------|---------------|------------|--------------|-------------------|
| 0               | 3             | n/a        | blocked      | none (first trigger) |
| 1               | 3             | < 90s      | blocked      | none |
| 1               | 3             | > 90s      | progressed   | hash-bound |
| 2               | 3             | < 90s      | blocked      | none |

| 3               | 3             | < 90s      | blocked      | none |

note: glob count > $hashbar AND mtime > 90s → hashless promise
