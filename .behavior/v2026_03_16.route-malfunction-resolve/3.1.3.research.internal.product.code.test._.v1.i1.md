**citation 4** — `blackbox/driver.route.drive.acceptance.test.ts:44-65`
```typescript
given('[case1] route bound with unpassed stones', () => {
  const scene = useBeforeAll(async () => {
    const tempDir = genTempDirForRhachet({
      slug: 'drive-case1',
      clone: ASSETS_DIR,
    });
    // setup...
    return { tempDir };
  });

  when('[t0] route.drive is invoked', () => {
    const result = useThen('route.drive succeeds', async () =>
      invokeRouteSkill({ skill: 'route.drive', args: {}, cwd: scene.tempDir }),
    );
    then('exit code is 0', () => {
      expect(result.code).toEqual(0);
    });
```

### relation to wish

the wish requires two test cases. this pattern structures tests with setup via useBeforeAll and result capture via useThen.
