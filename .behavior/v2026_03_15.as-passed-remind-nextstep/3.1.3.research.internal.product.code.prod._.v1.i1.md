## pattern.1: stepRouteStoneSet orchestration

**file:** `src/domain.operations/route/stepRouteStoneSet.ts`

**what:** orchestrates the `--as` dispatch for `passed`, `approved`, `promised`, `rewound`

**relation to wish:**
- controls which handler is called based on `--as` value
- not the place to add the reminder (handlers are responsible for output)

**citations:**

[1] dispatch to setStoneAsPassed (lines 68-87):
```typescript
if (input.as === 'passed') {
  const result = await setStoneAsPassed(
    {
      stone: input.stone,
      route: input.route,
    },
    context,
  );

  // if passed successfully, clear drive blocker state (progress made)
  if (result.passed) {
    await delDriveBlockerState({ route: input.route });
  }

  return {
    passed: result.passed,
    refs: result.refs,
    emit: result.emit,
  };
}
```

**verdict:** [REUSE] — no changes needed

## pattern.3: formatRouteStoneEmit output generation

**file:** `src/domain.operations/route/formatRouteStoneEmit.ts`

**what:** formats route stone operation output with tree structure

**relation to wish:**
- generates the CLI output string
- THIS IS WHERE THE REMINDER SHOULD BE ADDED
- needs to append reminder when `action === 'passed'` AND `passage === 'allowed'`

**citations:**

[4] guarded stone delegation (lines 151-161):
```typescript
if (input.operation === 'route.stone.set') {
  // delegate to formatGuardTree for full guard tree output
  if (input.action === 'passed' && input.guard) {
    const tree = formatGuardTree({
      stone: input.stone,
      passage: input.passage,
      note: input.note ?? null,
      reason: input.reason ?? null,
      guard: input.guard,
    });
    return [header, '', tree].join('\n');
  }
```

[5] unguarded stone output (lines 272-289):
```typescript
} else if (input.action === 'passed') {
  // format passage with optional note inline (passed without guard or selfReview)
  const passageValue = input.note
    ? `${input.passage} (${input.note})`
    : input.passage;

  if (
    (input.passage === 'blocked' || input.passage === 'malfunction') &&
    input.reason
  ) {
    lines.push(`   ├─ stone = ${input.stone}`);
    lines.push(`   ├─ passage = ${passageValue}`);
    lines.push(`   └─ reason = ${input.reason}`);
  } else {
    lines.push(`   ├─ stone = ${input.stone}`);
    lines.push(`   └─ passage = ${passageValue}`);
  }
}
```

**verdict:** [EXTEND] — add reminder after output for `passage === 'allowed'`
