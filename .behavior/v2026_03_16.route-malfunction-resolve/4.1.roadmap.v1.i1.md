## implementation summary

### fix 1: malfunction detection (stepRouteDrive.ts)

**what was done:**
```ts
// group by stone, find latest status per stone
const latestByStone = new Map<string, (typeof passageReports)[0]>();
for (const report of passageReports) {
  latestByStone.set(report.stone, report); // later entries overwrite earlier
}

// check if any stone's latest status is malfunction
const malfunctionReport = [...latestByStone.values()].find(
  (r) => r.status === 'malfunction',
);
```

**behavioral change:** a subsequent 'passed' status now resolves a prior malfunction

### fix 2: review cache exclusion (runStoneGuardReviews.ts)

```ts
// filter out malfunctions - only cache successful reviews
const cachedReviews = priorArtifacts.reviews.filter(
  (r) => r.exitClass !== 'malfunction',

```

**behavioral change:** malfunctions are not cached; fixed reviewers re-run
