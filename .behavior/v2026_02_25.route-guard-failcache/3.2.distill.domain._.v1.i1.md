## composition

### cache lookup flow

```
runStoneGuardJudges(input)
    │
    ├── enumerate prior judge artifacts via glob
    │
    ├── for each prior artifact:
    │       │
    │       ├── parse content (passed, reason)
    │       │
    │       └── if passed AND index valid AND not seen:
    │               └── add to priorByIndex cache map
    │
    └── for each judge command:
            │
            ├── check priorByIndex for cache hit
            │
            ├── if hit: return cached result
            │
            └── if miss: execute command, write artifact
```

### key modification

**before:** `if (index > 0 && !priorByIndex.has(index))`

**after:** `if (index > 0 && !priorByIndex.has(index) && passed)`
