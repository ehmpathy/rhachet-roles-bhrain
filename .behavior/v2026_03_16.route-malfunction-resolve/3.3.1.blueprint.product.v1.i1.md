## contracts

### fix 1: latest status detection

```ts
// before
const malfunction = passageEntries.find(e => e.status === 'malfunction');
if (malfunction) halt();

// after
const latestPerStone = getLatestStatusPerStone(passageEntries);
const unresolvedMalfunction = latestPerStone.find(e => e.status === 'malfunction');
if (unresolvedMalfunction) halt();
```

### fix 2: cache filter

```ts
// before
const cached = priorReviews.find(r => r.hash === artifactHash);

// after
const cached = priorReviews.find(r =>
  r.hash === artifactHash &&
  r.exitClass === 'passed'
);
```
