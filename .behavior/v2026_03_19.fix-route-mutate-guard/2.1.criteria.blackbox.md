# 2.1.criteria.blackbox

## usecase.1 = artifact writes to bound route

```
given('route bound at .route/xyz/')
  when('driver writes to .route/xyz/artifact.md')
    then('write is allowed')
      sothat('drivers can create artifacts in their bound route')
  when('driver writes to .route/xyz/subdir/artifact.md')
    then('write is allowed')
      sothat('drivers can organize artifacts in subdirectories')
  when('driver writes to .route/xyz/blocker/defect1.md')
    then('write is allowed')
      sothat('blocker explanations are visible alongside artifacts')
```

## usecase.2 = metadata writes to bound route blocked

```
given('route bound at .route/xyz/')
  when('driver writes to .route/xyz/.route/passage.jsonl')
    then('write is blocked')
      sothat('passage state cannot be corrupted')
  when('driver writes to .route/xyz/.route/.bind.branch.flag')
    then('write is blocked')
      sothat('bind state cannot be corrupted')
  when('driver writes to .route/xyz/.route/other-metadata.txt')
    then('write is blocked')
      sothat('all metadata is protected')
```

## usecase.3 = stone and guard protection unchanged

```
given('route bound at .route/xyz/')
  when('driver reads .route/xyz/2.criteria.stone')
    then('read is blocked')
      sothat('drivers cannot peek ahead at future stones')
  when('driver reads .route/xyz/2.criteria.guard')
    then('read is blocked')
      sothat('drivers cannot game review criteria')
  when('driver writes to .route/xyz/1.vision.stone')
    then('write is blocked')
      sothat('stone instructions cannot be modified')
```

## usecase.4 = behavior routes work identically

```
given('route bound at .behavior/xyz/')
  when('driver writes to .behavior/xyz/artifact.md')
    then('write is allowed')
      sothat('behavior routes work the same as .route/ routes')
  when('driver writes to .behavior/xyz/.route/passage.jsonl')
    then('write is blocked')
      sothat('metadata protection works for behavior routes')
  when('driver reads .behavior/xyz/2.criteria.stone')
    then('read is blocked')
      sothat('stone protection works for behavior routes')
```

## usecase.5 = privilege grants bypass protection

```
given('route bound at .route/xyz/')
  given('privilege granted via rhx route.mutate grant allow')
    when('driver writes to .route/xyz/.route/passage.jsonl')
      then('write is allowed')
        sothat('privileged operations can modify metadata')
    when('driver reads .route/xyz/2.criteria.stone')
      then('read is allowed')
        sothat('privileged operations can read stones')
```

## usecase.6 = no bound route allows all

```
given('no route bound to current branch')
  when('driver writes to .route/xyz/artifact.md')
    then('write is allowed')
      sothat('unbound branches have no restrictions')
  when('driver writes to .route/xyz/.route/passage.jsonl')
    then('write is allowed')
      sothat('guard only activates when route is bound')
```

## usecase.7 = blocker location

```
given('route bound at .route/xyz/')
  when('system writes blocker explanation')
    then('blocker is written to .route/xyz/blocker/*.md')
      sothat('blockers are visible alongside artifacts')
    then('blocker is NOT written to .route/xyz/.route/blocker/*.md')
      sothat('blockers are not hidden in metadata')
```

---

## boundary conditions

| condition | behavior |
|-----------|----------|
| path is within bound route, not in .route/ subdirectory | allowed |
| path is within bound route, in .route/ subdirectory | blocked |
| path is *.stone in bound route | blocked (read and write) |
| path is *.guard in bound route | blocked (read and write) |
| path is outside bound route | allowed (not checked by this guard) |
| privilege is granted | all operations on bound route allowed |
| no route bound | all operations allowed |

---

## error experience

| scenario | error message |
|----------|---------------|
| blocked by guard | shows route path, blocked path, and guidance to use route.drive |
| blocked by guard (bash) | shows truncated command and reason |

---

## out of scope

- cross-route writes (writes to a route other than the bound one) — not checked by this guard
- nested routes (routes inside routes) — not supported
- blocker migration — no consumers exist, no migration needed
