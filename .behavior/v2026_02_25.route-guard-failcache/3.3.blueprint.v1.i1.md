## contracts

### cache filter condition

**before:**
```ts
if (index > 0 && !priorByIndex.has(index)) {
  priorByIndex.set(index, new RouteStoneGuardJudgeArtifact({...}));
}
```

**after:**
```ts
if (index > 0 && !priorByIndex.has(index) && passed) {
  priorByIndex.set(index, new RouteStoneGuardJudgeArtifact({...}));
}
```

### behavior contract

| exit code | cache on lookup? |
|-----------|------------------|
| 0 (pass) | yes |
| 2 (block) | no |
| 1 (error) | no |
| other | no |
