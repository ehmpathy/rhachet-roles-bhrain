# blackbox criteria: route guard failcache

> **approach: cache successes only**
>
> only cache exit 0 (success). all other exit codes retry each time.
> simpler mental model: "cache the pass, retry everything else"

## usecase.1 = judge passes on first attempt

```
given('a stone with a guard')
  when('the judge executes and passes (exit 0)')
    then('the stone passage is allowed')
      sothat('the route can proceed')
    then('subsequent attempts skip the judge')
      sothat('we avoid redundant work')
    then('output shows "cached" indicator')
      sothat('user knows the result was remembered')
```

## usecase.2 = judge blocks on first attempt (explicit constraint)

```
given('a stone with a guard')
  when('the judge executes and blocks (exit 2)')
    then('the stone passage is blocked')
      sothat('the constraint is enforced')
    then('the block reason is displayed')
      sothat('user knows what to address')
    then('subsequent attempts re-run the judge')
      sothat('constraint can clear when addressed')
    then('output shows NO "cached" indicator')
      sothat('user knows this will retry')
```

## usecase.3 = judge fails on first attempt (unexpected error)

```
given('a stone with a guard')
  when('the judge executes and fails (exit 1)')
    then('the stone passage is blocked')
      sothat('we dont proceed on broken state')
    then('the error output is displayed')
      sothat('user can diagnose the issue')
    then('subsequent attempts re-run the judge')
      sothat('transient errors get a fresh chance')
    then('output shows NO "cached" indicator')
      sothat('user knows this will retry')
```

## usecase.4 = judge blocks then constraint addressed

```
given('a stone with a guard')
  given('the judge blocked on a prior attempt (exit 2)')
  given('the constraint has been addressed (e.g., approval granted)')
  when('the judge executes again')
    then('the judge runs fresh (not cached)')
      sothat('it can detect the addressed constraint')
  when('the judge passes (exit 0)')
    then('the stone passage is allowed')
      sothat('the route can proceed')
    then('the success is cached')
      sothat('future attempts skip the judge')
```

## usecase.5 = judge fails then succeeds on retry

```
given('a stone with a guard')
  given('the judge failed on a prior attempt (exit 1)')
  when('the judge executes again')
    then('the judge runs fresh (not cached)')
      sothat('the transient error can clear')
  when('the judge passes (exit 0)')
    then('the stone passage is allowed')
      sothat('the route can proceed after recovery')
    then('the success is cached')
      sothat('future attempts skip the judge')
```

## usecase.6 = judge passes then artifact changes

```
given('a stone with a guard')
  given('the judge passed on a prior attempt (exit 0)')
  when('the artifact content changes')
    then('the judge runs fresh (cache miss)')
      sothat('the new content is evaluated')
  when('the judge blocks (exit 2)')
    then('the stone passage is blocked')
      sothat('the new content respects constraints')
```

## usecase.7 = multiple judges on same stone

```
given('a stone with multiple judges')
  when('all judges pass (exit 0)')
    then('the stone passage is allowed')
      sothat('all constraints are satisfied')
    then('each passed judge is cached independently')
      sothat('only changed judges re-run')

  when('one judge fails (exit 1) and others pass')
    then('the stone passage is blocked')
      sothat('we dont proceed on broken state')
    then('the failed judge is NOT cached')
      sothat('it retries on next attempt')
    then('the passed judges ARE cached')
      sothat('we dont redo successful work')

  when('one judge blocks (exit 2) and others pass')
    then('the stone passage is blocked')
      sothat('constraint is enforced')
    then('the blocked judge is NOT cached')
      sothat('it retries when constraint addressed')
    then('the passed judges ARE cached')
      sothat('we dont redo successful work')
```

## usecase.8 = cache corruption

```
given('a stone with a guard')
  given('the cache file is corrupted or unreadable')
  when('the stone is evaluated')
    then('the judge runs fresh')
      sothat('corrupted cache doesnt block progress')
    then('no error is surfaced about the cache')
      sothat('user experience is seamless')
```

---

## boundary conditions

| condition | input | expected output |
|-----------|-------|-----------------|
| exit code 0 | judge passes | cached, passage allowed |
| exit code 2 | judge blocks | NOT cached, passage blocked |
| exit code 1 | judge errors | NOT cached, passage blocked |
| exit code 127 | command not found | NOT cached, passage blocked |
| exit code 137 | killed (OOM) | NOT cached, passage blocked |
| any other exit | unexpected | NOT cached, passage blocked |
| artifact unchanged + prior pass | same hash, cached | cache hit, skip judge |
| artifact unchanged + prior non-pass | same hash, not cached | fresh judge |
| artifact changed | new hash | cache miss, fresh judge |
| cache file absent | first run | fresh judge |
| cache file corrupted | invalid json | fresh judge |
