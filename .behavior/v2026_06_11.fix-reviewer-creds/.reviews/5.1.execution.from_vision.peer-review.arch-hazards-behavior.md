# blocker.1: read-modify-write without protection on temp file

**rule**: .agent/repo=bhuild/role=behaver/briefs/practices/behavior.execution/architect/rule.forbid.behavior-hazards.md

**locations**:
- blackbox/review.join-intersect.acceptance.test.ts
- blackbox/review.join-intersect.acceptance.test.ts

Multiple tests perform read-modify-write on a file in a temp directory to create a diff. This matches the race condition anti-pattern (get then set without lock/transaction). If the temp dir generation reuses a slug on retry (common in repeatably-wrapped tests), or if test infrastructure runs in parallel, the append can produce different file content on each execution. This leads to non-deterministic test outcomes and intermittent failures.

**snippet**:
```ts
const dirtyPath = path.join(tempDir, 'src/dirty.ts');
const dirtyContent = await fs.readFile(dirtyPath, 'utf-8');
await fs.writeFile(
  dirtyPath,
  dirtyContent + '\n// modified for test\n',
  'utf-8',
);
```

Different invocation methods (node, ts-node, jest, npx, eval) produce different argv layouts, causing silent misparsing of --brain, --refs, --paths etc. This leads to unexpected defaults (e.g. wrong brain or missing diffs) and different runtime behavior depending on how the CLI is launched.

const args = argv.slice(skipCount).filter((arg) => arg !== '--');
// ... later in loop

# nitpick.1: non-atomic latest-log selection

**rule**: .agent/repo=bhuild/role=behaver/briefs/practices/behavior.execution/architect/rule.forbid.behavior-hazards.md

**locations**:
- blackbox/review.join-intersect.acceptance.test.ts
- blackbox/review.join-intersect.acceptance.test.ts

After invoking the review skill, code does readdir + sort + pop to locate the newest log entry. If the skill writes logs asynchronously, the dir is empty, or multiple runs overlap, this picks the wrong file or crashes on !. This is a time/order assumption hazard that can cause flaky test assertions.

**snippet**:
```ts
const logDirs = await fs.readdir(path.join(tempDir, '.log/bhrain/review'));
const latestLog = logDirs.sort().pop()!;
const scopeJson = await fs.readFile(
  path.join(tempDir, '.log/bhrain/review', latestLog, 'input.scope.json'),
  'utf-8',
);
```
