# research: internal product code (prod)

## pattern 1: execSync for git commands [REPLACE]

**location**: `src/domain.operations/reflect/savepoint/setSavepoint.ts:94-109`

**current pattern**:
```typescript
// [1] line 94-97: get HEAD commit hash
const commitHash = execSync('git rev-parse HEAD', {
  cwd: input.scope.gitRepoRoot,
  encoding: 'utf-8',
}).trim();

// [2] line 100-103: get staged diff (ENOBUFS here)
const stagedPatch = execSync('git diff --staged', {
  cwd: input.scope.gitRepoRoot,
  encoding: 'utf-8',
});

// [3] line 106-109: get unstaged diff (ENOBUFS here too)
const unstagedPatch = execSync('git diff', {
  cwd: input.scope.gitRepoRoot,
  encoding: 'utf-8',
});
```

**problem**: `execSync` buffers stdout in node. default `maxBuffer` is 1MB. large diffs exceed this.

**replacement**: shell redirect for [2] and [3]. [1] is fine (tiny output).
