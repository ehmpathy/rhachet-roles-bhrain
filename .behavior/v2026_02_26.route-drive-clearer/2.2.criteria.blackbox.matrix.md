| approved | agent | ✓ passes | progresses to next stone |
| not approved | agent | ✗ blocks | stays on current stone |

## combined view

all independent variables across usecases:

| ind: command | ind: caller | ind: approval state | dep: outcome |
|--------------|-------------|---------------------|--------------|
| `--as approved` | agent | n/a | blocked, redirect to human |
| `--as approved` | human | n/a | succeeds |
| `--as passed` | agent | not approved | blocked, shows hook guidance |
| `--as passed` | agent | approved | passes |

**gaps**: none

**decomposition signal**: 3 independent variables is manageable. no decomposition needed.
