1. **unit test (formatGuardTree.test.ts or formatRouteStoneEmit.test.ts):**
   - add cases for `passage === 'allowed'` with guard
   - add cases for `passage === 'allowed'` unguarded
   - assert `result.toContain('the way continues')`
   - assert `result.toContain('rhx route.drive')`

2. **acceptance test (driver.route.set.acceptance.test.ts):**
   - extend case1 (unguarded passed) and case3 (guarded passed) with reminder assertion
   - add negative assertion to case4 (blocked) — no reminder shown
