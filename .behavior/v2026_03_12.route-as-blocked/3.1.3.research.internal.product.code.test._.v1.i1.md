### pattern.4: createArticulationFile helper [REUSE]

**location**: `src/domain.operations/route/promise/getSelfReviewChallengeDecision.test.ts:13-28`

**quote**:
```typescript
const createArticulationFile = async (input: {
  route: string;
  stone: string;
  index: number;
  slug: string;
}): Promise<void> => {
  const articulationDir = path.join(input.route, 'review', 'self');
  await fs.mkdir(articulationDir, { recursive: true });
  await fs.writeFile(
    path.join(articulationDir, `${input.stone}.${input.index}.${input.slug}.md`),
    '# self-review\n',
  );
};
```

**implication**: create similar helper for blocked articulation file:
```typescript
const createBlockerArticulationFile = async (input: {
  route: string;
  stone: string;
}): Promise<void> => {
  const articulationDir = path.join(input.route, '.route', 'blocker');

    path.join(articulationDir, `${input.stone}.md`),
    '# blocker\n\n## what blocks me\n\n## what i tried\n\n## what i need\n',
  );
};
```

**action**: REUSE pattern, new helper.

### pattern.7: backdateTriggeredReport helper [REUSE]

**location**: `src/domain.operations/route/stepRouteStoneSet.test.ts:146-166`

**quote**:
```typescript
const backdateTriggeredReport = async (input: {
  stone: string;
  slug: string;
}): Promise<void> => {
  const routeDir = path.join(tempDir, '.route');
  const files = await fs.readdir(routeDir).catch(() => []);
  const sinceFile = files.find(
    (f) =>
      f.includes(`${input.stone}.guard.selfreview.${input.slug}`) &&
      f.endsWith('.triggered.since'),
  );
  if (sinceFile) {
    const filepath = path.join(routeDir, sinceFile);
    const mtimePast = new Date(Date.now() - 91 * 1000);
    await fs.utimes(filepath, mtimePast, mtimePast);
  }
};
```

**implication**: note - blocked flow does NOT need time-based backdate. blocked challenge only checks:
1. triggered file exists
2. articulation file exists

no time enforcement in blocked flow.

**action**: NOT NEEDED for blocked.
