## domain operations

### extant operations (extended)

| operation | change | description |
|-----------|--------|-------------|
| stepRouteDrive | EXTEND | add latest-status-per-stone logic |
| runStoneGuardReviews | EXTEND | filter malfunctions from cache |
| getAllPassageReports | REUSE | returns all passage entries in order |
| getAllStoneGuardArtifactsByHash | REUSE | returns prior artifacts for hash |

### operation changes

**stepRouteDrive (hook mode):**
```typescript
// before: find ANY malfunction
const malfunctionReport = passageReports.find(r => r.status === 'malfunction');

// after: find LATEST status per stone, then check for malfunction
const latestByStone = new Map<string, PassageReport>();
for (const report of passageReports) {
  latestByStone.set(report.stone, report); // later overwrites earlier
}
const malfunctionReport = [...latestByStone.values()].find(
  r => r.status === 'malfunction'
);
```

**runStoneGuardReviews:**
```typescript
// before: reuse all prior reviews for hash
const doneIndices = new Set(priorArtifacts.reviews.map(r => r.index));

// after: filter out malfunctions before reuse
const cachedReviews = priorArtifacts.reviews.filter(
  r => r.exitClass !== 'malfunction'
);
const doneIndices = new Set(cachedReviews.map(r => r.index));
```

## relationships

### treestruct

```
passage.jsonl
  └── PassageReport[]
        └── stone → status (latest wins)

review artifacts
  └── RouteStoneGuardReviewArtifact[]
        └── hash + exitClass → cacheable?
```

### dependencies

```
stepRouteDrive
  └── getAllPassageReports
        └── reads passage.jsonl

runStoneGuardReviews
  └── getAllStoneGuardArtifactsByHash
        └── reads .route/*.guard.review.*.md
```

## composition for wish fulfillment

### flow 1: malfunction recovery

```
1. getAllPassageReports() → [{ stone: X, status: 'malfunction' }, { stone: X, status: 'passed' }]
2. group by stone, take latest → { stone: X, status: 'passed' }
3. find malfunction → null (none with latest = malfunction)
4. hook proceeds (exit 0 or 2)
```

### flow 2: review cache exclusion

```
1. getAllStoneGuardArtifactsByHash() → [{ exitClass: 'malfunction', index: 1 }]
2. filter r.exitClass !== 'malfunction' → []
3. doneIndices = empty set
4. reviewer runs fresh (index 1 not in done set)
```
