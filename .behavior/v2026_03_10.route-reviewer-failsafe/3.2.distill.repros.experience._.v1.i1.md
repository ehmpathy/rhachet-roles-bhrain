## reproduction feasibility

### reviewer output capture

**test utilities available:**
- `genTempDir` for isolated temp directory
- `useBeforeAll` for shared scene setup
- echo commands to simulate reviewer output
- snapshot tests for output format

**setup required:**
- temp directory with stone file
- guard file with reviewer command that exits with specific code
- capture of stdout/stderr from `route.stone.set`

**test sketch:**
```ts
given('[case1] reviewer exits with code 1 (malfunction)', () => {
  const scene = useBeforeAll(async () => {
    const tempDir = genTempDir({ slug: 'reviewer-malfunction' });
    await fs.writeFile(path.join(tempDir, '1.test.stone'), '# Test');
    await fs.writeFile(
      path.join(tempDir, '1.test.guard'),
      [
        'reviews:',
        '  - bash -c \'echo "stdout content"; echo "stderr content" >&2; exit 1\'',
      ].join('\n'),
    );
    return { tempDir };
  });

  when('[t0] route.stone.set runs', () => {
    const result = useThen('stone set completes', async () =>
      stepRouteStoneSet({ stone: '1.test', route: scene.tempDir, as: 'passed' }, noopContext),
    );

    then('artifact contains stdout in tree bucket', async () => {
      const artifactPath = path.join(scene.tempDir, '.route', '1.test.guard.review.*.md');
      const artifact = await fs.readFile(artifactPath, 'utf-8');
      expect(artifact).toContain('├─ stdout');
      expect(artifact).toContain('stdout content');
    });

    then('artifact contains stderr in tree bucket', () => {
      expect(artifact).toContain('├─ stderr');
      expect(artifact).toContain('stderr content');
    });

    then('artifact shows blocked by malfunction', () => {
      expect(artifact).toContain('blocked by malfunction');
      expect(artifact).toContain('exit code: 1');
    });
  });
});
```

### malfunction escalation

**test utilities available:**
- `stepRouteDrive` for hook mode behavior
- fixture with malfunction state in passage.jsonl

**setup required:**
- temp directory with route
- passage.jsonl with `status: "malfunction"` entry

**test sketch:**
```ts
given('[case3] malfunction recorded in passage.jsonl', () => {
  const scene = useBeforeAll(async () => {
    const tempDir = genTempDir({ slug: 'drive-malfunction' });
    await fs.mkdir(path.join(tempDir, '.route'), { recursive: true });
    await fs.writeFile(
      path.join(tempDir, '.route', 'passage.jsonl'),
      JSON.stringify({ stone: '1.test', status: 'malfunction' }) + '\n',
    );
    return { tempDir };
  });

  when('[t0] route.drive runs', () => {
    const result = useThen('drive completes', async () =>
      stepRouteDrive({ route: scene.tempDir, mode: 'hook' }),
    );

    then('returns exit code 3', () => {
      expect(result.emit?.stderr?.code).toBe(3);
    });

    then('tells driver to halt', () => {
      expect(result.emit?.stdout).toContain('malfunction');
      expect(result.emit?.stdout).toContain('human must fix');
    });
  });
});
```

**test sketch:**

```ts
describe('formatTreeBucket', () => {
  it('formats content in tree bucket', () => {
    const result = formatTreeBucket({
      label: 'stdout',
      content: 'line 1\nline 2',
    });

    expect(result).toMatchSnapshot();
    expect(result).toContain('├─ stdout');
    expect(result).toContain('│  ├─');
    expect(result).toContain('│  │  line 1');
    expect(result).toContain('│  │  line 2');
    expect(result).toContain('│  └─');
  });
});
```
