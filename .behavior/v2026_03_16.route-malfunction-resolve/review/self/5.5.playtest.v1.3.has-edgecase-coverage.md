### why this coverage is sufficient

the playtest doesn't need to test every permutation because:

1. **latest-wins is symmetric**
   - malfunction → pass tested
   - the logic uses `reduce` to find latest
   - any sequence with latest=passed unblocks

2. **cache exclusion is binary**
   - malfunction excluded: tested
   - non-malfunction included: tested
   - no gradation between these states

3. **acceptance tests are end-to-end**
   - 14 tests exercise real CLI flows
   - test fixtures simulate real scenarios
   - no mocks that could hide edge cases
