## output flow

```
1. clone runs: route.stone.set --stone X --as passed

2. stdout (immediate):
   🦉 so you're sayin' there's a chance?

3. progress (to stderr, rendered inline):
   ✓ j1: inflight
   ✗ j1: finished 0.8s

4. stdout (tree):
   🗿 route.stone.set
      ├─ stone = X
      ├─ passage = blocked
      ├─ reason = wait for human approval
      └─ guard
         └─ judges
             └─ j1: rhx route.stone.judge --mechanism approved? ...
                 ├─ finished [TIME] ✗
                 └─ reason: wait for human approval

5. stderr (judge artifact content):
   🔎 judge 1
      ├─
      │
      │  passed: false
      │  reason: wait for human approval
      │
      │  ✋ halted, human approval required
      │     ├─ please ask your human to
      │     │  └─ rhx route.stone.set --stone X --as approved
      │     │
      │     └─ after human approves, run
      │        └─ rhx route.stone.set --stone X --as passed
      │
      └─

6. exit code: 2 (constraint block)
```
