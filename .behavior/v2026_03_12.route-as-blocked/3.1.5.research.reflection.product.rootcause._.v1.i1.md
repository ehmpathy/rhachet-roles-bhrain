# root cause: route --as blocked

## what is the symptom?

- **what is broken or absent?**: robots have no explicit way to signal they are stuck and need human help
- **how does it manifest?**: robots either sit idle, make bad guesses, or hallucinate through obstacles
- **when does it occur?**: when robot encounters a genuine blocker — design decision, access gap, domain expertise gap

---

## 5 whys tree

### branch 1: robot wastes time when stuck

1. why? → robot has no command to signal "i am stuck"
2. why? → route.stone.set only supports `passed`, `approved`, `promised`, `rewound`
3. why? → no one implemented `--as blocked` yet
4. why? → the need was not obvious until robots started to drive behavior routes
5. why? → **root cause: route.stone.set was built for forward progress, not escalation**

### branch 2: human cannot tell if robot is stuck

1. why? → no blocked status in route.drive output
2. why? → passage.jsonl has no blocked entry for that stone
3. why? → robot could not record blocked status
4. why? → no command to do so
5. why? → **root cause: same as branch 1 — absent `--as blocked` command**

### branch 3: route.drive hook prompts forever

1. why? → hook does not know robot is stuck
2. why? → blocked status is not recorded
3. why? → no command to record it
4. why? → **root cause: same as branch 1**

---

## root cause summary

| layer | issue | is root cause? |
|-------|-------|----------------|
| symptom | robot wastes time, human confused | no |
| proximate | no blocked status in passage.jsonl | no |
| deeper | no command to record blocked | no |
| **root** | route.stone.set built for progress, not escalation | **yes** |

---

## recommendation

**address root cause**: add `--as blocked` to route.stone.set

this is the correct layer because:
1. it adds the absent command
2. it records to passage.jsonl (makes hook aware)
3. it surfaces in route.drive (makes human aware)
4. it gives robot explicit escalation path (makes robot less stuck)

no symptom-level fix would be durable. to address the root cause is straightforward — extend extant patterns.
