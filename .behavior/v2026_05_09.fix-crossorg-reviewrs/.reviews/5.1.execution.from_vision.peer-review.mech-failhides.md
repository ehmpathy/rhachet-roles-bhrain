The try/catch in isDirectory catches all errors from fs.stat and silently returns false, hiding potential errors. This violates failhide rule as it should fail fast instead of hiding errors. Change to not catch errors or use allowlist for specific errors and rethrow others.

**snippet**:
```ts
const isDirectory = async (input: {
  pattern: string;
  cwd: string;
}): Promise<boolean> => {
  try {
    const fullPath = path.isAbsolute(input.pattern)
      ? input.pattern
      : path.join(input.cwd, input.pattern);
    const stat = await fs.stat(fullPath);
    return stat.isDirectory();
  } catch {
    return false;
  }
};
```
