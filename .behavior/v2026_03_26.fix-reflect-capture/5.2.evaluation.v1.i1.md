## codepath tree (as implemented)

### setSavepoint.ts

```
setSavepoint
├─ [○] generateTimestamp() — retained
├─ [○] get HEAD commit hash via execSync — retained (small output)
│
├─ [~] get staged diff
│  ├─ [+] if mode === 'apply': shell redirect to file
│  │     └─ execSync(`git diff --staged > "${stagedPatchPath}"`, { cwd })
│  └─ [+] if mode === 'plan': no diff capture (hash/size via pipes)
│
├─ [~] get unstaged diff
│  ├─ [+] if mode === 'apply': shell redirect to file
│  │     └─ execSync(`git diff > "${unstagedPatchPath}"`, { cwd })
│  └─ [+] if mode === 'plan': no diff capture (hash/size via pipes)
│
├─ [~] compute hash
│  ├─ [+] if mode === 'apply': sha256sum on files
│  │     └─ execSync(`cat "${staged}" "${unstaged}" | (sha256sum || shasum -a 256)`)
│  └─ [+] if mode === 'plan': sha256sum via pipe
│        └─ execSync(`(git diff --staged; git diff) | (sha256sum || shasum -a 256)`)
│
├─ [~] compute byte sizes
│  ├─ [+] if mode === 'apply': fs.statSync(path).size
│  └─ [+] if mode === 'plan': wc -c via pipe
│        └─ execSync(`git diff --staged | wc -c`)
│
├─ [○] construct paths — retained
├─ [~] write files if apply mode
│  ├─ [-] fs.writeFileSync(stagedPatchPath, stagedPatch) — deleted (shell redirect writes)
│  ├─ [-] fs.writeFileSync(unstagedPatchPath, unstagedPatch) — deleted (shell redirect writes)
│  └─ [○] fs.writeFileSync(commitPath, commitHash) — retained (small, no buffer issue)
│
└─ [○] return Savepoint — retained, interface unchanged
```
