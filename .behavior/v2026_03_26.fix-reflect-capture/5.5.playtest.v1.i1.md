# playtest: fix reflect.snapshot capture ENOBUFS

## prerequisites

- node.js installed
- bash shell
- `rhx` command available (via `npx rhachet roles link`)

note: the playtest creates its own temp git repo — no need to be in a repo

## sandbox

all file operations target `@gitroot/.temp/playtest-reflect/`

---

## happy paths

### 1. small diff (baseline)

```bash
# create temp directory
mkdir -p .temp/playtest-reflect
cd .temp/playtest-reflect

# init git repo
git init
git config user.email "test@test.com"
git config user.name "Test"

# initial commit
echo "init" > init.txt
git add init.txt
git commit -m "initial"

# create small staged change
echo "small change" > small.txt
git add small.txt

# capture savepoint
cd ../..
rhx reflect.savepoint set --cwd .temp/playtest-reflect
```

**expected outcome:**
- command succeeds (exit 0)
- output shows `🌕 reflect.savepoint set`
- output shows `staged.patch = [SIZE]ytes`

### 2. large diff (the fix)

```bash
# create large staged change (>1MB)
cd .temp/playtest-reflect
for i in {1..15000}; do echo "line $i: the quick brown fox jumps over the lazy dog repeated many times" >> large.txt; done
git add large.txt

# capture savepoint
cd ../..
rhx reflect.savepoint set --cwd .temp/playtest-reflect
```

**expected outcome:**
- command succeeds (exit 0) — NOT ENOBUFS
- output shows `staged.patch = [SIZE]ytes` where SIZE > 1000000

### 3. apply mode

```bash
rhx reflect.savepoint set --cwd .temp/playtest-reflect --mode apply
```

**expected outcome:**
- command succeeds
- output shows `✨ savepoint captured`
- files written to `~/.rhachet/storage/...`

## edgey paths

### empty staged diff

```bash
cd .temp/playtest-reflect
git reset HEAD large.txt  # unstage the large file
cd ../..
rhx reflect.savepoint set --cwd .temp/playtest-reflect
```

**expected behavior:**
- command succeeds
- output shows `staged.patch = 0ytes`

### both staged and unstaged

```bash
cd .temp/playtest-reflect
git add large.txt  # re-stage
echo "unstaged change" >> unstaged.txt  # create unstaged
cd ../..
rhx reflect.savepoint set --cwd .temp/playtest-reflect
```

**expected behavior:**
- command succeeds
- output shows both `staged.patch` and `unstaged.patch` with sizes

## pass/fail criteria

| check | pass | fail |
|-------|------|------|
| small diff | exits 0, shows output | any error |
| large diff (>1MB) | exits 0, no ENOBUFS | ENOBUFS error |
| apply mode | files written | any error |
| empty diff | exits 0, shows 0ytes | any error |

**summary:**

- ✓ pass if: all 4 checks succeed, especially large diff (>1MB) without ENOBUFS
- ✗ fail if: any command shows `Error: spawnSync /bin/sh ENOBUFS`
