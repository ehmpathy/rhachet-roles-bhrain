# rule: no-gerunds

## severity: blocker

forbid gerunds (-ing words used as nouns)

gerunds weaken prose by adding unnecessary syllables and passive voice implications

## examples

bad:
- "the running of the bulls"
- "his speaking was unclear"
- "writing is hard"

good:
- "the bulls ran"
- "he spoke unclearly"
- "to write is hard"
