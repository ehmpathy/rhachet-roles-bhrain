## pattern 1: passage status detection via getAllPassageReports

**[EXTEND]** — need to change from "any malfunction" to "latest status per stone"

### current implementation

**citation 1** — `src/domain.operations/route/stepRouteDrive.ts:50-62`
```typescript
if (input.mode === 'hook') {
  const passageReports = await getAllPassageReports({ route });

  // group by stone, find latest status per stone
  const latestByStone = new Map<string, (typeof passageReports)[0]>();
  for (const report of passageReports) {
    latestByStone.set(report.stone, report); // later entries overwrite earlier
  }

  // check if any stone's latest status is malfunction
  const malfunctionReport = [...latestByStone.values()].find(
    (r) => r.status === 'malfunction',
  );
```

### relation to wish

the wish requires that "if the last status for the stone in passage.jsonl is 'passed', then the malfunction was fixed". this pattern groups by stone and finds the latest status, so the last entry wins.
