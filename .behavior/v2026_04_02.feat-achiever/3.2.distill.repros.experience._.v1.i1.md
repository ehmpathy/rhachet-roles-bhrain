# distill: experience reproductions — achiever role

## experience reproductions

| experience | entry point | user actions | expected outcome | test type |
|------------|-------------|--------------|------------------|-----------|
| triage on session end | hook.onStop | brain attempts to end session | session halted, uncovered asks shown | acceptance |
| goal creation | goal.memory.set CLI | brain provides full Goal schema | goal persisted, coverage recorded | integration |
| goal retrieval | goal.memory.get CLI | brain queries goals | goals returned with status | integration |
| triage query | goal.infer.triage CLI | brain checks uncovered asks | uncovered asks and extant goals shown | integration |
| ask accumulation | hook.onTalk | peer sends message | ask appended to inventory | integration |

## journey test sketches

### journey 1: multi-part request triage

this is the primary flow from the vision (t0-t9).

#### step table

| step | action | user sees |
|------|--------|-----------|
| t0 | before any changes | empty .goals/ directory |
| t1 | peer sends multi-part message | hook.onTalk fires, ask accumulated |
| t2 | session ends | hook.onStop fires, brain halted, triage prompt shown |
| t3 | brain runs triage | uncovered asks shown with hashes |
| t4 | brain creates first goal (full schema) | goal persisted, coverage recorded |
| t5 | brain creates second goal | goal persisted, coverage recorded |
| t6 | brain verifies coverage | zero uncovered, session may continue |

#### bdd structure

```typescript
given('[case1] multi-part request triage journey')
  when('[t0] before any changes')
    then('.goals/ is empty or absent')
    then('asks.inventory.jsonl is empty or absent')
    then('asks.coverage.jsonl is empty or absent')
    then('state matches snapshot')

  when('[t1] peer sends multi-part message')
    then('ask is appended to asks.inventory.jsonl')
    then('ask has content hash')
    then('inventory matches snapshot')

  when('[t2] session ends (onStop hook)')
    then('hook detects uncovered asks')
    then('triage prompt is shown')
    then('output matches snapshot')

  when('[t3] brain runs goal.infer.triage')
    then('uncovered asks are shown')
    then('extant goals are shown (empty)')
    then('output matches snapshot')

  when('[t4] brain creates first goal')
    then('goal file is created')
    then('status flag file is created')
    then('coverage entry is appended')
    then('goal matches snapshot')

  when('[t5] brain creates second goal')
    then('second goal file is created')
    then('second coverage entry is appended')
    then('goal matches snapshot')

  when('[t6] brain verifies coverage')
    then('goal.infer.triage shows zero uncovered')
    then('output matches snapshot')
```

##### t4 goal creation (snapshot target)

**input:**
```bash

  purpose: ci to pass before merge
  benefit: unblocks the pr
```

## critical paths

| critical path | description | why critical |
|---------------|-------------|--------------|
| triage on session end | brain is halted until all asks covered | this is the core promise — no ask lost |
| goal creation with full schema | brain must articulate why/what/how | forced foresight is the differentiation |
| coverage verification | brain sees zero uncovered → may continue | proof that triage worked |

these are the "golden paths" — if any of these fail, the product fails.

## ergonomics review

| journey | input ergonomics | output ergonomics | friction notes |
|---------|------------------|-------------------|----------------|
| triage query | natural — just run `goal.infer.triage` | natural — treestruct shows uncovered | none |
| goal creation | awkward — full YAML via stdin | natural — treestruct shows what was persisted | consider YAML file input option |
| goal retrieval | natural — just run `goal.memory.get` | natural — treestruct shows goals | none |

### friction: goal creation via stdin

**problem:** brain must pipe full YAML via stdin or heredoc.

**mitigation options:**
1. accept `--file path/to/goal.yaml` in addition to stdin
2. accept individual flags like `--slug`, `--ask`, `--purpose` (verbose but explicit)
3. for v1, stdin is acceptable — brain can construct heredoc

**decision for v1:** stdin is acceptable. brain is capable of heredoc. file input can be added later if friction is high.

## reproduction feasibility

### test utilities available

| utility | source | purpose |
|---------|--------|---------|
| genTempDir | test-fns | isolated test directory |
| genTempDirForRhachet | local | rhachet-linked test directory |
| given/when/then | test-fns | BDD structure |
| useThen | test-fns | share operation results |
| invokeRouteSkill | local | invoke skill via shell |

### setup required

1. create temp directory with `.goals/$branch/` structure
2. optionally pre-populate with goal fixtures
3. invoke skills via shell (acceptance) or functions (integration)
4. assert on file contents and CLI output

### test file names

journey tests use `.play.acceptance.test.ts` suffix to match repo's acceptance runner:

| journey | file name |
|---------|-----------|
| multi-part request triage | `goal.triage.play.acceptance.test.ts` |
| goal lifecycle | `goal.lifecycle.play.acceptance.test.ts` |

### test sketch

```typescript
// goal.triage.play.acceptance.test.ts
describe('achiever multi-part triage journey', () => {
  given('[case1] multi-part request triage', () => {
    const scene = useBeforeAll(async () => {
      const tempDir = genTempDirForRhachet({ slug: 'achiever-triage' });
      return { tempDir };
    });

    when('[t0] before any changes', () => {
      then('.goals/ is empty', async () => {
        const goalsDir = path.join(scene.tempDir, '.goals');
        expect(await fs.pathExists(goalsDir)).toBe(false);
      });
    });

    when('[t3] brain runs goal.infer.triage', () => {
      const result = useThen('triage executes', async () => {
        return invokeSkill({
          skill: 'goal.infer.triage',
          args: ['--scope', 'repo'],
          cwd: scene.tempDir,
        });
      });

      then('output shows uncovered asks', () => {
        expect(result.stdout).toContain('uncovered asks');
      });

      then('output matches snapshot', () => {
        expect(result.stdout).toMatchSnapshot();
      });
    });
  });
});
```
