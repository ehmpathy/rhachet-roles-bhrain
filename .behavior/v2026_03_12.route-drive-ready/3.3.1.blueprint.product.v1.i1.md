# blueprint: `--as arrived` for route.drive

## summary

implement `--as arrived` alias and drum nudge for stuck clones:

1. **alias**: `--as arrived` maps to `--as passed` in `stepRouteStoneSet`
2. **drum nudge**: after 7+ hooks (reuse extant blocker state count), add zen quote before command prompt, show both options
3. **output text**: change `--as passed` to `--as arrived` in drive output

## state file format

no new state file needed. reuse extant blocker state:

```
.route/.drive.blockers.latest.json
{
  "count": 3,
  "stone": "1.vision"
}
```

note: `count` already tracks hooks without passage (increments on hook, resets on `--as passed`). use `count >= 7` for nudge threshold.
