## domain operations

### classifyExitCode - NEW

```ts
export const classifyExitCode = (input: { code: number }): ExitCodeClass => {
  if (input.code === 0) return 'passed';
  if (input.code === 2) return 'constraint';
  return 'malfunction';
};
```

### formatTreeBucket - NEW

```ts
export const formatTreeBucket = (input: {
  label: string;
  content: string;
}): string => {
  const lines: string[] = [];
  lines.push(`├─ ${input.label}`);
  lines.push(`│  ├─`);
  lines.push(`│  │`);
  for (const line of input.content.split('\n')) {
    lines.push(`│  │  ${line}`);
  }
  lines.push(`│  │`);
  lines.push(`│  └─`);
  return lines.join('\n');
};
```


## composition

### usecase.1 = reviewer output capture

```
runStoneGuardReviews
  → execa command
  → capture { stdout, stderr, exitCode }
  → classifyExitCode(exitCode)
  → formatTreeBucket(stdout) + formatTreeBucket(stderr)
  → write artifact
  → return GuardExecutionResult
```

### usecase.2 = judge output capture

```
runStoneGuardJudges
  → execa command
  → capture { stdout, stderr, exitCode }
  → classifyExitCode(exitCode)
  → formatTreeBucket(stdout) + formatTreeBucket(stderr)
  → write artifact
  → return GuardExecutionResult
```

### usecase.3 = malfunction escalation

```
route.stone.set --as passed
  → runStoneGuardJudges
  → if exitClass === 'malfunction':
      → setPassageReport({ status: 'malfunction' })

route.drive (hook mode)
  → read passage.jsonl
  → if status === 'malfunction':
      → emit escalation message
      → exit code 3 (not retry)
```

### usecase.4 = CLI output

```
route.stone.set
  → for each reviewer:
      → emit formatReviewerOutput (🔎 with tree buckets)
  → for each judge:
      → emit formatJudgeOutput (🪶 with tree buckets)
```
