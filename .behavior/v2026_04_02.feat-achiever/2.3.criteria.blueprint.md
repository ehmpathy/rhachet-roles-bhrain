## subcomponent contracts

### skills

```
given('goal.infer.triage skill contract')
  then('exposes: goal.infer.triage --from peer|self --context route|repo')
  then('returns: uncovered asks, extant goals, coverage entries')
  then('reads: asks.inventory.jsonl, asks.coverage.jsonl, .goals/*.goal.yaml')
  then('writes: none (read-only query)')

given('goal.memory.set skill contract')
  then('exposes: goal.memory.set --scope route|repo --covers $hash,... [goal yaml via stdin]')
  then('exposes: goal.memory.set --slug $slug --status $choice --scope route|repo [--covers $hash,...]')
  then('returns: path to persisted goal, covered hashes')
  then('writes: .goals/$branch/$offset.$slug.goal.yaml')
  then('writes: .goals/$branch/$offset.$slug.status=$choice.flag')
  then('writes: asks.coverage.jsonl (appends coverage entries)')
  then('throws: schema error when goal schema is incomplete')
  then('throws: error when goal not found (update mode)')
  then('throws: error when branch is main (repo scope)')

given('goal.memory.get skill contract')
  then('exposes: goal.memory.get --scope route|repo [--status $choice]')
  then('returns: list of goals (may be empty)')
  then('reads: .goals/*.goal.yaml, .goals/*.status=*.flag')
```

### hooks

```
given('hook.onTalk handler contract')
  then('fires: when peer sends message')
  then('writes: appends ask to asks.inventory.jsonl with content hash')
  then('outputs: reminder to consider goal impact')
  then('does not halt: brain continues work')

given('hook.onStop handler contract')
  then('fires: when session ends')
  then('reads: asks.inventory.jsonl, asks.coverage.jsonl')
  then('halts: until all asks are covered')
  then('outputs: prompt to run goal.infer.triage')
  then('yields to: goal.infer.triage (route.drive is silent if triage needed)')
```

### domain objects

```
given('Goal domain object contract')
  then('schema: slug, why{ask,purpose,benefit}, what{outcome}, how{task,gate}, status{choice,reason}, when?, source, createdAt, updatedAt')
  then('status.choice: blocked | enqueued | inflight | fulfilled')
  then('source: peer:human | peer:robot | self')
  then('when: optional, one of {goal: slug} or {event: description}')
  then('validates: all required fields present')
  then('validates: status.choice is valid enum value')

given('Ask domain object contract')
  then('schema: hash, content, receivedAt')
  then('hash: deterministic content hash')
  then('persisted as: jsonl line in asks.inventory.jsonl')

given('Coverage domain object contract')
  then('schema: hash, goalSlug, coveredAt')
  then('persisted as: jsonl line in asks.coverage.jsonl')
```

## composition boundaries

```
given('triage flow composition')
  then('hook.onTalk accumulates asks to inventory')
  then('hook.onStop reads inventory and coverage to detect uncovered')
  then('goal.infer.triage provides the uncovered asks for brain to process')
  then('goal.memory.set persists goals and records coverage')
  then('cycle repeats until all asks are covered')

given('persistence composition')
  then('goals are files: .goals/$branch/$offset.$slug.goal.yaml')
  then('status is files: .goals/$branch/$offset.$slug.status=$choice.flag')
  then('inventory is file: asks.inventory.jsonl (append-only)')
  then('coverage is file: asks.coverage.jsonl (append-only)')
  then('route-scoped uses: $route/.goals/')
  then('repo-scoped uses: .goals/$branch/')

given('scope resolution composition')
  then('--scope route requires brain to be within a route')
  then('--scope repo requires branch != main')
  then('skills determine scope location before read/write')
```
