  then('level defaults to 1 when absent')

### ReviewerBudgetState

```
given('ReviewerBudgetState contract')
  then('tracks { slug, budgetUsed, budgetTotal, level, status }')
  then('status is one of: queued | active | approved | exhausted')
  then('persists in route scope (not per stone)')
```

### route.guard.budget skill

```
given('route.guard.budget skill contract')
  then('accepts: --for review --add N')
  then('accepts: --for review --peer $slug --add N')
  then('increments budget for target reviewers')
  then('transitions exhausted reviewers back to active')
```

## composition boundaries

### review execution flow

```
given('review execution composes budget tracker')
  then('runStoneGuardReviews reads reviewer budget state')
  then('runStoneGuardReviews groups reviewers by level')
  then('runStoneGuardReviews executes level N before level N+1')
  then('level N+1 waits until all level N reviewers are terminal')
  then('terminal means: approved OR exhausted')
```

### budget consumption flow

```
given('budget consumption on review')
  then('budget decrements only on successful review exit')
  then('budget does not decrement on malfunction')
  then('budget does not decrement on cached review')
  then('reviewer transitions to exhausted when budget reaches 0')
```

### exhaustion halt flow

```
given('exhaustion triggers route halt')
  then('after all reviewers terminal, check if any exhausted')
  then('if any exhausted, block stone with blocker=review.peer.exhausted')
  then('halt prompts human with extend-or-approve options')
  then('human approval satisfies both exhaustion and explicit approved? judge')
```

### rewind reset flow

```
given('rewind resets exhaustion')
  then('--as rewound resets reviewer budget to full for that route')
  then('--as rewound transitions exhausted reviewers to queued')
```

## integration boundaries

given('passage.jsonl integration')

  then('budget consumption happens only when review command executes')
