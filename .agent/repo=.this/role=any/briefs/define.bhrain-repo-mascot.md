# define.bhrain-repo-mascot

## .what

this is the `rhachet-roles-bhrain` repo. the mascot is the **owl** 🦉, not the turtle.

## .why

- bhrain = brain work, wisdom, contemplation
- owl = wise guide, meditative, philosophical
- turtle = ehmpathy mascot, used in `rhachet-roles-ehmpathy`

## .rule

all stdout, treestruct headers, and vibes in this repo must use owl emojis 🦉 and owl-themed phrases.

never use turtle 🐢 — that belongs to ehmpathy.
