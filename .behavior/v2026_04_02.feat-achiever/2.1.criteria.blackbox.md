# blackbox criteria: achiever role

## episode experience

### usecase.1 = multi-part request triage

the primary flow: human sends message with multiple asks, brain triages all into tracked goals.

```
given(a brain with achiever role loaded)
  given(human sends message with multiple asks)
    when(hook.onTalk fires)
      then(each ask is appended to asks.inventory.jsonl with content hash)
        sothat(no ask is lost)
      then(brain receives reminder to consider goal impact)
        sothat(brain can triage inline if needed)

    when(session ends and hook.onStop fires)
      then(brain is halted until triage complete)
        sothat(no ask escapes without coverage)

    when(brain runs goal.infer.triage)
      then(uncovered asks are shown with their hashes)
        sothat(brain knows what needs coverage)
      then(extant goals are shown for reference)
        sothat(brain can link asks to extant goals)

    when(brain creates goal via goal.memory.set with --covers)
      then(goal is persisted with full schema)
        sothat(goal has why, what, how articulated)
      then(coverage entry is appended to asks.coverage.jsonl)
        sothat(ask is marked as covered)

    when(all asks are covered)
      then(goal.infer.triage shows zero uncovered)
        sothat(brain knows triage is complete)
      then(session may end)
        sothat(work can continue)
```

### usecase.2 = goal lifecycle

tracking a goal from creation through completion.

```
given(a goal exists in .goals/)
  when(brain starts work on goal)
    then(goal status can be updated to inflight)
      sothat(active work is visible)

  when(brain completes goal)
    then(goal status can be updated to fulfilled)
      sothat(completion is recorded)
    then(status.reason contains verification evidence)
      sothat(completion is auditable)

  when(goal is blocked on another goal)
    then(goal status can be updated to blocked)
      sothat(dependency is explicit)
    then(when.goal contains blocking goal slug)
      sothat(blocker is traceable)
```

### usecase.3 = goal persistence across context

goals survive context compression.

```
given(goals exist in .goals/)
  when(context is compressed)
    then(goals remain in filesystem)
      sothat(goals survive long sessions)

  when(new session starts)
    then(brain can read extant goals via goal.memory.get)
      sothat(work can resume)
```

### usecase.4 = self-generated goals

brain notices an item that should be a goal and creates it from self-observation.

```
given(brain observes an item that should be a goal)
  when(brain runs goal.memory.set with source=self)
    then(goal is persisted with source=self)
      sothat(origin is distinguishable from peer asks)
```

### usecase.5 = partial goals (quick capture)

brain can capture goals incrementally without full schema upfront via CLI flags.

**interface:** partial goals use CLI flags, not YAML stdin.

```
# full goal: YAML via stdin (all fields required)
cat goal.yaml | rhx goal.memory.set --scope repo

# partial goal: CLI flags (only --slug required)
rhx goal.memory.set --scope repo \
  --slug 'fix-auth-test' \
  --why.ask 'fix the flaky test'

# partial goal with multiple fields
rhx goal.memory.set --scope repo \
  --slug 'fix-auth-test' \
  --why.ask 'fix the flaky test' \
  --why.purpose 'ci needs to pass' \
  --what.outcome 'test passes reliably'

# flag values: 'string' | @stdin | @stdin.N (null-separated)
echo 'fix the flaky test' | rhx goal.memory.set --scope repo \
  --slug 'fix-auth-test' \
  --why.ask @stdin

# multiple stdin values via null separator
printf 'fix the test\0ci needs to pass' | rhx goal.memory.set --scope repo \
  --slug 'fix-auth-test' \
  --why.ask @stdin.0 \
  --why.purpose @stdin.1
```

```
given(brain wants to capture a goal quickly)
  when(brain runs goal.memory.set with CLI flags)
    then(goal is persisted with supplied fields only)
      sothat(brain can collect thoughts incrementally)
    then(goal is marked as incomplete)
      sothat(triage can remind brain to finish articulation)

given(incomplete goals exist)
  when(brain runs goal.infer.triage)
    then(incomplete goals are shown separately from complete goals)
      sothat(brain knows which goals need further articulation)
    then(incomplete goals list which fields are absent)
      sothat(brain knows what to fill in)

given(brain wants to complete a partial goal)
  when(brain runs goal.memory.set with --slug and additional field flags)
    then(goal is updated with new fields)
      sothat(goal becomes more complete over time)
    then(goal completeness is recalculated)
      sothat(completion status reflects current state)
```

## exchange experience

### contract.1 = goal.infer.triage

atomic: show uncovered asks and extant goals.

```
given(asks.inventory.jsonl contains peer input)
  given(asks.coverage.jsonl contains some coverage entries)
    when(goal.infer.triage is invoked)
      then(uncovered asks are returned)
        sothat(brain knows what needs triage)
      then(each uncovered ask includes content hash)
        sothat(brain can reference ask by hash)
      then(extant goals are returned)
        sothat(brain can link to extant)
      then(coverage entries are returned)
        sothat(brain sees what is already covered)

given(asks.inventory.jsonl is empty)
  when(goal.infer.triage is invoked)
    then(zero uncovered asks are returned)
      sothat(brain knows no asks need triage)

given(all asks are covered)
  when(goal.infer.triage is invoked)
    then(zero uncovered asks are returned)
      sothat(brain knows triage is complete)

given(incomplete goals exist)
  when(goal.infer.triage is invoked)
    then(incomplete goals are returned separately)
      sothat(brain sees which goals need further articulation)
    then(each incomplete goal lists its absent fields)
      sothat(brain knows what to fill in)

given(all goals are complete)
  when(goal.infer.triage is invoked)
    then(incomplete goals list is empty)
      sothat(brain knows all goals are fully articulated)
```

### contract.2 = goal.memory.set (new goal)

atomic: create goal with full schema (YAML stdin) or partial schema (CLI flags).

**two modes:**
1. **full goal mode:** YAML via stdin — all required fields must be present
2. **partial goal mode:** CLI flags — only --slug required, other fields optional

**CLI flags for partial goals:**
```
--slug 'string'           # required: goal identifier
--why.ask 'string'        # what was said
--why.purpose 'string'    # the motivation
--why.benefit 'string'    # what success enables
--what.outcome 'string'   # the desired end state
--how.task 'string'       # actionable next step
--how.gate 'string'       # verification criteria
--source 'string'         # peer:human | peer:robot | self
--status 'string'         # blocked | enqueued | inflight | fulfilled
--status.reason 'string'  # why this status
--covers 'hash1,hash2'    # ask hashes this goal covers
```

**flag value sources:**
- `'literal string'` — direct value
- `@stdin` — read entire stdin as value
- `@stdin.N` — read Nth null-separated value from stdin (0-indexed)

```
given(no goal exists with slug)
  when(goal.memory.set is invoked with full YAML via stdin)
    then(goal file is created at .goals/$branch/$offset.$slug.goal.yaml)
      sothat(goal is persisted)
    then(status flag file is created at .goals/$branch/$offset.$slug.status=$choice.flag)
      sothat(status is visible from filename)
    then(coverage entries are appended to asks.coverage.jsonl)
      sothat(asks are marked as covered)
    then(path and covered hashes are returned)
      sothat(caller knows result)

given(no goal exists with slug)
  when(goal.memory.set is invoked with CLI flags (--slug required, others optional))
    then(goal file is created with supplied fields only)
      sothat(quick capture is possible)
    then(goal includes meta.complete = false)
      sothat(incompleteness is tracked)
    then(goal includes meta.absent = list of absent fields)
      sothat(brain knows what to fill in later)

given(goal has zero fields besides slug)
  when(goal.memory.set is invoked with only --slug)
    then(goal is created with meta.complete = false)
      sothat(even minimal goals are accepted for quick capture)

given(slug is absent)
  when(goal.memory.set is invoked without --slug and without YAML with slug)
    then(error is returned)
      sothat(every goal must have an identifier)

given(both YAML stdin and CLI flags are provided)
  when(goal.memory.set is invoked)
    then(CLI flags override YAML values)
      sothat(flags take precedence for targeted updates)
```

### contract.3 = goal.memory.set (update goal)

atomic: update extant goal status.

```
given(goal exists with slug)
  when(goal.memory.set is invoked with --slug and --status)
    then(status flag filename is updated)
      sothat(new status is visible)
    then(status.reason is updated in goal.yaml)
      sothat(reason is recorded)
    then(updatedAt is updated)
      sothat(change time is recorded)

given(goal exists with slug)
  when(goal.memory.set is invoked with --slug and --covers)
    then(coverage entries are appended)
      sothat(additional asks are marked as covered)
```

### contract.4 = goal.memory.get

atomic: retrieve extant goals.

```
given(goals exist in .goals/)
  when(goal.memory.get is invoked)
    then(all goals are returned)
      sothat(brain has full picture)

given(goals exist with various statuses)
  when(goal.memory.get is invoked with --status inflight)
    then(only inflight goals are returned)
      sothat(brain can filter by status)

given(no goals exist)
  when(goal.memory.get is invoked)
    then(empty list is returned)
      sothat(brain knows no goals exist)
```

### contract.5 = asks.inventory accumulation

atomic: hook.onTalk accumulates asks.

```
given(hook.onTalk fires with peer input)
  when(ask is accumulated)
    then(ask is appended to asks.inventory.jsonl)
      sothat(ask is recorded)
    then(ask entry includes content hash)
      sothat(ask is identifiable)
    then(asks remain in order of receipt)
      sothat(sequence is preserved)
```

### contract.6 = scope (route vs repo)

atomic: goals are scoped correctly.

```
given(brain is within a route)
  when(goal.memory.set is invoked with --scope route)
    then(goal is persisted to $route/.goals/)
      sothat(goal is route-scoped)

given(brain is not within a route)
  when(goal.memory.set is invoked with --scope repo)
    then(goal is persisted to .goals/$branch/)
      sothat(goal is repo-scoped)

given(brain is on main branch)
  when(goal.memory.set is invoked with --scope repo)
    then(error is returned)
      sothat(goals on main are forbidden)
```
