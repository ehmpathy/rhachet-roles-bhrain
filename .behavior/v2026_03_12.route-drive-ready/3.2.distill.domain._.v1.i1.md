# distillation: domain

## usecases

1. clone runs `--as arrived` → system treats as `--as passed`
2. route.drive shows hook counter and drum nudge after 7 hooks
3. hook counter resets when clone attempts `--as arrived` or `--as passed`

## domain objects

### entities

none new required. extant entities:
- `DriveBlockerState` — tracks consecutive stop blocks (reuse, don't extend)

### literals

| literal | shape | purpose |
|---------|-------|---------|
| `DriveHookState` | `{ hooksWithoutAttempt: number }` | track hooks without passage attempt |

### events

none new required.

## domain operations

### new operations

| operation | input | output | purpose |
|-----------|-------|--------|---------|
| `getDriveHookState` | `{ route, stone }` | `DriveHookState` | read hook counter |
| `setDriveHookState` | `{ route, stone }` | `{ state }` | increment hook counter |
| `delDriveHookState` | `{ route, stone }` | void | reset hook counter on attempt |

### extended operations

| operation | change | purpose |
|-----------|--------|---------|
| `stepRouteStoneSet` | add `'arrived'` to `as` union, map to `'passed'` | alias support |
| `stepRouteDrive` | call `setDriveHookState`, check threshold, format nudge | hook counter + nudge |
| `formatRouteDrive` | change `--as passed` to `--as arrived` in output | new command text |

### new formatters

| formatter | purpose |
|-----------|---------|
| `formatRouteDriveNudge` | drum nudge output (hooks >= 7) |

## relationships

```
stepRouteDrive
  ├─ getDriveHookState (read counter)
  ├─ setDriveHookState (increment on hook)
  ├─ formatRouteDrive (normal, hooks < 7)
  └─ formatRouteDriveNudge (nudge, hooks >= 7)

stepRouteStoneSet
  ├─ (extant) setStoneAsPassed
  ├─ (extant) delDriveBlockerState
  └─ (new) delDriveHookState (reset counter on attempt)
```

## composition

1. **hook flow:**
   - route.drive fires
   - `setDriveHookState` increments counter
   - if counter < 7: `formatRouteDrive` (normal)
   - if counter >= 7: `formatRouteDriveNudge` (drum nudge)

2. **passage flow:**
   - clone runs `--as arrived`
   - `stepRouteStoneSet` maps `arrived` → `passed`
   - `delDriveHookState` resets counter
   - extant pass flow proceeds

