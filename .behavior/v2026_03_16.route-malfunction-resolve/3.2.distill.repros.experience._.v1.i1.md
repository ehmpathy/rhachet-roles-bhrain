## journey test sketches

### journey 1: malfunction recovery

```
given('[case1] malfunction then pass unblocks hook')
  when('[t0] route.drive hook mode with only malfunction in passage')
    then('exit code is 3 (malfunction escalate)')
    then('stderr mentions malfunction')
  when('[t1] passed status is appended after malfunction')
    then('exit code is 0 (hook allows stop)')
    then('passage.jsonl has both entries')
```

| step | action | user sees |

## reproduction feasibility

### test utilities available

- `genTempDirForRhachet` — creates temp dir with git and symlinks
- `invokeRouteSkill` — invokes skills via shell, captures exit code
- `fs.writeFile` / `fs.appendFile` — seed passage.jsonl state
- mock reviewer — controllable via flag files

### setup required

1. clone fixture with mock reviewer
2. link driver role
3. create feature branch
4. bind route
5. seed initial state (malfunction in passage or reviewer flag)

### concrete test sketch

```typescript
given('[case1] malfunction then pass unblocks hook', () => {
  const scene = useBeforeAll(async () => {
    const tempDir = genTempDirForRhachet({ slug: 'malfunction-case1', clone: ASSETS_DIR });
    await execAsync('npx rhachet roles link --role driver', { cwd: tempDir });
    await execAsync('git checkout -b vlad/test-malfunction', { cwd: tempDir });
    await invokeRouteSkill({ skill: 'route.bind.set', args: { route: '.' }, cwd: tempDir });
    // seed malfunction
    await fs.mkdir(path.join(tempDir, '.route'), { recursive: true });
    await fs.writeFile(path.join(tempDir, '.route', 'passage.jsonl'),
      '{"stone":"1.feature","status":"malfunction"}\n');
    return { tempDir };
  });

  when('[t0] hook with malfunction', () => {
    const result = useThen('route.drive halts', async () =>
      invokeRouteSkill({ skill: 'route.drive', args: { mode: 'hook' }, cwd: scene.tempDir })
    );
    then('exit code is 3', () => expect(result.code).toEqual(3));
  });

  when('[t1] passed appended', () => {
    const result = useThen('append passed and drive', async () => {
      await fs.appendFile(path.join(scene.tempDir, '.route', 'passage.jsonl'),
        '{"stone":"1.feature","status":"passed"}\n');
      return invokeRouteSkill({ skill: 'route.drive', args: { mode: 'hook' }, cwd: scene.tempDir });
    });
    then('exit code is 0', () => expect(result.code).toEqual(0));
  });
});
```
