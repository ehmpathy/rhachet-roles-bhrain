# blueprint criteria: route guard failcache

## blackbox criteria satisfied

- usecase.1 = judge passes on first attempt ✓
- usecase.2 = judge blocks on first attempt ✓
- usecase.3 = judge fails on first attempt ✓
- usecase.4 = judge blocks then constraint addressed ✓
- usecase.5 = judge fails then succeeds on retry ✓
- usecase.6 = judge passes then artifact changes ✓
- usecase.7 = multiple judges on same stone ✓
- usecase.8 = cache corruption ✓

## subcomponent contracts

### judge cache

```
given('judge cache contract')
  then('exposes: getJudgeResult(input: { cacheKey: string }) => JudgeResult | null')
    sothat('we can check for cached pass')
  then('exposes: setJudgeResult(input: { cacheKey: string, result: JudgeResult }) => void')
    sothat('we can cache successful results')
  then('returns null when cache file absent')
    sothat('first runs execute fresh')
  then('returns null when cache file corrupted')
    sothat('corrupted cache triggers fresh execution')
  then('returns null when cache file unreadable')
    sothat('permission issues trigger fresh execution')
```

### cache key generator

```
given('cache key generator contract')
  then('exposes: getJudgeCacheKey(input: { stone, mechanism, artifactHash }) => string')
    sothat('cache keys are deterministic and content-addressed')
  then('produces different keys for different artifact hashes')
    sothat('artifact changes invalidate cache')
  then('produces same key for same inputs')
    sothat('cache hits work correctly')
```

### judge executor

```
given('judge executor contract')
  then('exposes: executeJudge(input: { command, stone, route }) => { exitCode, stdout, stderr }')
    sothat('we can run judge commands')
  then('captures exit code accurately')
    sothat('we can distinguish pass (0), block (2), error (1/other)')
  then('captures stdout and stderr')
    sothat('we can display judge output to user')
  then('does not throw on non-zero exit')
    sothat('caller can handle exit codes')
```

## composition boundaries

### stone passage evaluation

```
given('stone passage evaluation')
  then('for each judge in stone guard:')
    then('generate cache key from artifact hash')
    then('check cache for prior pass')
    then('if cache hit with pass: skip execution, use cached result')
    then('if cache miss or prior non-pass: execute judge fresh')
    then('if judge passes (exit 0): cache the result')
    then('if judge does not pass (exit != 0): do NOT cache')
  then('passage allowed only if ALL judges pass')
  then('passage blocked if ANY judge does not pass')
```

### cache write decision

```
given('cache write decision')
  then('write to cache ONLY when exit code == 0')
    sothat('only successes are remembered')
  then('do NOT write to cache when exit code == 2')
    sothat('blocks retry when constraint addressed')
  then('do NOT write to cache when exit code == 1 or other')
    sothat('errors retry when transient issue clears')
```
