## the one gotcha to recall

the `rhx` wrapper prints a `🪨 run solid skill` banner to **stdout** on every skill run
(rhachet `invokeRun.js:54`, unconditional `console.log`). a statusLine command must emit
only the bare line, so the shipped config invokes the renderer directly via `node -e`
(not `rhx`) to bypass the banner.
