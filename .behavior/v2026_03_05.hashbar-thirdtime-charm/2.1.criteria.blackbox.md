# blackbox criteria: 30s cooldown, self-trust after one change

## usecase.1 = cooldown is 30s

```
given('self-review triggered')
  when('clone attempts to promise immediately')
    then('promise is rejected')
      sothat('clone must wait before promise accepted')
  when('clone waits 30s and attempts to promise')
    then('promise is accepted')
      sothat('30s cooldown is enforced')
```

## usecase.2 = default hashbar is 1

```
given('self-review with no hashbar specified')
  when('clone is challenged, edits once (count=2 > hashbar=1), waits 30s')
    then('promise is accepted')
    then('trigger is now hashless (self-trust)')
      sothat('one edit reaches self-trust with default hashbar=1')
  when('clone edits again, waits only 10s (less than 30s from prior trigger)')
    then('promise is accepted')
      sothat('self-trust means timer persists across edits')
```

## usecase.3 = hash change resets timer before hashbar

```
given('self-review at first attempt (count <= hashbar)')
  when('clone waits 20s, edits artifact, immediately attempts to promise')
    then('promise is rejected')
      sothat('edit resets the 30s timer')
  when('clone waits 30s after edit, attempts to promise')
    then('promise is accepted')
      sothat('timer starts fresh after each edit')
```

## usecase.4 = hash change does not reset timer after hashbar (self-trust)

```
given('self-review with count > hashbar (trigger is hashless)')
  when('clone edits artifact')
    then('timer does not reset')
      sothat('self-trust is active')
  when('clone waits rest of time from prior trigger, attempts to promise')
    then('promise is accepted')
      sothat('hashless trigger persists across hash changes')
```

## usecase.5 = explicit hashbar overrides default

```
given('self-review with hashbar: 3 specified in guard')
  when('clone edits once, waits 20s, edits again, immediately attempts')
    then('promise is rejected')
      sothat('count <= hashbar means timer resets on edit')
  when('clone edits 3 times total, waits 20s, edits again, immediately attempts')
    then('promise is accepted')
      sothat('count > hashbar means self-trust, timer persists')
```

## usecase.6 = no edit required to promise

```
given('self-review triggered')
  when('clone does not edit artifact')
    when('clone waits 30s and attempts to promise')
      then('promise is accepted')
        sothat('edit is not forced, only cooldown')
```
