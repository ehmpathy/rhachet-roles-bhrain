# blackbox criteria: expand $route in artifact globs

## usecase.1 = guard artifact patterns with $route

```
given('guard with $route in artifact pattern')
  when('behaver runs route')
    then('bhrain expands $route to actual route path')
      sothat('artifacts at .behavior/xyz/... are found')
    then('glob runs from repo root')
      sothat('path resolution is consistent')
    then('guard proceeds with found artifacts')
      sothat('route continues without "artifact not found" errors')

given('guard with $route present multiple times')
  when('behaver runs route')
    then('all $route instances are expanded')
      sothat('complex patterns like $route/sub/$route-ref.md work')

given('guard with no $route in artifact pattern')
  when('behaver runs route')
    then('pattern is used as-is from repo root')
      sothat('absolute patterns still work')

given('guard artifact pattern with no matches')
  when('behaver runs route')
    then('empty array is returned')
      sothat('guard can handle "no artifacts" case')
```

given('stone with no guard artifacts defined')
