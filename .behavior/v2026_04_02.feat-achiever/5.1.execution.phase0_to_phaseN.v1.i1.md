### onTalk hook workaround

rhachet's Role.build() hooks abstraction only supports:
- onBoot → SessionStart
- onTool → PreToolUse
- onStop → Stop

rhachet does not map onTalk → UserPromptSubmit.

**solution:** implemented via init executables that add UserPromptSubmit hook directly to settings.json:

```
src/domain.roles/achiever/inits/
├── init.claude.sh              # main entrypoint
├── init.claude.hooks.sh        # adds UserPromptSubmit hook to settings.json
└── claude.hooks/
    └── userpromptsubmit.ontalk.sh  # hook that runs on user message
```

**to install the hook:**
```sh
npx rhachet roles init --role achiever
```

this adds to `.claude/settings.json`:
```json
{
  "hooks": {
    "UserPromptSubmit": [{
      "matcher": "*",
      "hooks": [{
        "type": "command",
        "command": "./node_modules/.bin/rhachet run --repo bhrain --role achiever --init claude.hooks/userpromptsubmit.ontalk",
        "timeout": 5,
        "author": "repo=bhrain/role=achiever"
      }]
    }]
  }
}
```
